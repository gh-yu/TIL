# HTML

- 웹페이지의 **구조**를 나타내기 위한 언어
- 태그로 구성되어 있다

### 태그


- 태그 : 꺾쇠 괄호로 표시, **시작태그**와 **종료태그**로 만들어짐
    - `<태그명> 내용 </태그명>`
    - `<h1>내용</h1>`
- 태그는 **속성명**과 **속성값**이 있을 수 있다.
    - `<태그명 속성명=”속성값”> 내용 </태그명>`
    - `<h1 id=”title”>내용</h1>`
- 크롤링에서 자주 사용되는 태그 종류
    
    
    | 태그명 | 역할 |
    | --- | --- |
    |  div | 구역 나누기 |
    | a | 링크 |
    | h1 | 제목 |
    | p | 문단 |
    | ul, li | 목록 |
- 태그는 **부모태그**와 **자식태그**가 있다.
