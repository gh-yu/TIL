# CSS 선택자

## CSS란?

- 웹사이트의 **디자인을 표시하기 위한 언어**
- 글자 색 변경, 폰트 크기, 가로 세로 길이 변경

## CSS 선택자란?

- 디자인을 변경할 **HTML 태그를 선택하는 것**
    
    **= 크롤링할 HTML 태그를 선택하는 것**
    

### CSS 선택자의 종류

- 태그 선택자
    - 태그의 이름으로 선택
    - ex. h1, a
- id 선택자
    - id 값으로 선택
    - ex. #articleBody
- class 선택자
    - class 값으로 선택
    - ex. .info_group
- 자식 선택자
    - 보통 내가 원하는 태그에 별명이 없을 때 사용
    - 바로 아래에 있는 태그를 선택
    - ex. .news_headline > h4
